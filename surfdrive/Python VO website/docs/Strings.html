Hoofdstuk 10 - Strings

In programmeeromgevingen worden teksten weergegeven door strings. Dit hoofdstuk geeft details over strings, en over functies die beschikbaar zijn om strings aan te pakken.

10.1 Herhaling omgaan met strings

In hoofdstuk 3 heb ik strings kort geïntroduceerd. Ik heb uitgelegd dat een string een tekst is, die omsloten is door enkele of dubbele aanhalingstekens, en dat een string iedere lengte mag hebben, inclusief nul tekens lang. Het hoofdstuk legde ook uit dat je twee strings aan elkaar kunt plakken met behulp van de +, en dat je een string zichzelf kunt laten herhalen door middel van de *. Bijvoorbeeld:

woord1 = "appel"
woord2 = ' banaan '
print( woord1 )
print( woord2 )
print( woord1 + woord2 )
print( 3 * woord1 )
print( woord2 * 3 )
print( 2 * woord1 + 2 * woord2 )

Ik gaf ook aan dat je de lengte van een string kunt bepalen met de len() functie. String vergelijkingen heb ik uitgelegd in hoofdstuk 6. In hoofdstuk 6 gaf ik ook aan dat de in operator gebruikt kan worden om te testen of tekens of substrings voorkomen in een string. Hoofdstuk 7 legde uit hoe je met een for loop alle tekens in een string kunt doorlopen.

woord1 = "mango"
woord2 = "banaan"

for letter in woord1:
    if letter in woord2:
        print( woord1, "en", woord2, "bevatten beide de letter", letter )

10.2.1 Strings over meerdere regels

In Python kunnen strings meerdere regels beslaan. Dat kan nuttig zijn wanneer je een erg lange string in je programma hebt, of als je de output van de string op een specifieke manier wilt formatteren.

De aanbevolen manier om strings over meerdere regels te spreiden in Python is het gebruik van drievoudige aanhalingstekens:

lange_string = """Ik heb mijn buik vol van te worden behandeld
als een schaap. Wat is de zin van op vakantie gaan als je
gewoon maar een toerist bent die wordt rondgereden in een bus ..."""

print( lange_string )

10.3.1 String indices

Ieder teken in een string heeft een positie, en die positie kun je weergeven door het index nummer van de positie. De indices beginnen bij 0 en lopen op tot aan de lengte van de string.

Bij het woord "python" zit 'p' op positie 0, 'y' op positie 1 en 'n' op positie.

De lengte van een string s kun je berekenen met len(s); de laatste letter van de string heeft dus index len(s)-1. De lengte van het woord "python" is 6, dus de laatste letter, 'n' zit op positie 5.

Als een string is opgeslagen in een variabele, dan kun je de individuele letters van de string krijgen via de variabele naam en de index van de gevraagde letter tussen vierkante haken ([]) rechts ernaast. Bijvoorbeeld:

woord = "aardbei"
print( woord[4] )
print( woord[2] )
print( woord[0] )

Dit drukt af:
b
r
a

10.3.2 Strings doorlopen

Ik heb eerder uitgelegd hoe je de tekens van een string kunt doorlopen middels een for loop:

woord = "appel"
for letter in woord:
    print( letter )

Nu je indices begrijpt, realiseer je je waarschijnlijk wel dat je die ook kunt gebruiken om een string te doorlopen.

Zo doe je het met een for loop:

woord = "appel"
for i in range( len( woord ) ):
    print( woord[i] )

En zo met een while loop:

woord = "appel"
i = 0
while i < len( woord ):
    print( woord[i], "- ", end="" )
    i += 1

10.4 string methodes

Er is een aantal methodes beschikbaar die ontworpen zijn om strings te bewerken. Al deze methodes worden toegepast op een string om een operatie uit te voeren. Omdat strings onveranderbaar zijn, zullen deze methodes nooit de string waarop ze werken wijzigen, maar ze retourneren in plaats daarvan een gewijzigde versie van de string.

10.4.1 strip()

strip() verwijdert spaties aan het begin en einde van een string, inclusief eventuele "newline" tekens en andere tekens die als spaties gezien kunnen worden. Als je iets anders dan spaties wilt verwijderen, kun je als parameter een string meegeven die bestaat uit alle te verwijderen tekens.

tekst = "   En nu iets heel anders \n    "
print( "[" + tekst + "]" )

tekst = tekst.strip()
print( "[" + tekst + "]" )

In dit voorbeeld gebruiken de '[' en ']' om duidelijk te laten zien welke tekens 'gestript' worden.

10.4.2 upper() en lower()

upper() creëert een versie van een string met alle letters als hoofdletters.
lower() werkt op dezelfde manier, maar maakt van alle letters kleine letters.
Geen van beide methodes heeft parameters.

tekst = "The Meaning of Life"
print( tekst )
print( tekst.upper() )
print( tekst.lower() ))

10.4.3 find()

find() kun je gebruiken om in een string te zoeken naar de start-index van een bepaalde substring. Als parameter krijgt de methode de gezochte substring.

Als de substring niet voorkomt, wordt -1 opgeleverd.

tekst = "Humpty Dumpty zat op de muur"
print( tekst.find( "zat" ) )
print( tekst.find( "t" ) )
print( tekst.find( "q" ) )

10.4.4 replace()

replace() vervangt alle instanties van een substring in een string door een andere substring. Als parameters krijgt het de substring die gezocht wordt, en de substring die als vervanging dient. Optioneel kan een derde, numerieke parameter meegegeven worden die aangeeft hoe vaak een vervanging moet plaatsvinden.

Ik wil hier nogmaals benadrukken dat strings onveranderbaar zijn, dus de replace() functie maakt niet echt vervangingen in de string; hij retourneert een nieuwe string die een kopie is van de originele string, waarbij de vervangingen zijn gemaakt.

tekst = " Humpty Dumpty zat op de muur "
print( tekst.replace( "zat op" , "viel van" ) )
