Hoofdstuk 12 - Lists

Een "list" (Engelse woord voor "lijst") is een geordende verzameling van data items, net zoals een tuple. Het verschil tussen lists en tuples is dat lists veranderbaar zijn. Dit maakt ze tot een zeer flexibele data structuur, waar je op veel manieren gebruik van kunt maken.

12.1 Basis van lists

Een list is een verzameling (of "collectie") elementen. De elementen van een list zijn geordend. Omdat ze geordend zijn, kun je ieder element van een list benaderen via een index, net zoals je de tekens in een string kunt benaderen. De indices beginnen bij nul, net als bij strings. In Python kun je lists herkennen aan het feit dat de elementen van een list tussen vierkante haken ([]) staan. Je kunt het aantal elementen in een list achterhalen door middel van de len() functie. Via een for loop kun je de elementen van de list doorlopen. Je mag data types in een list mixen. Je kunt de functies max(), min() en sum() gebruiken op een list. Je kunt testen of een element voorkomt in een list via de in operator (of het niet voorkomen van een element via de not in operator).

Lijst doorlopen met een while loop

Soms wil je dat iets in de code herhaald wordt, bijvoorbeeld als je voor elk element uit een lijst iets wilt doen. Je bekijkt dan het eerste element uit de lijst, doet iets, vervolgens de tweede, doet weer het zelfde, enzovoort totdat je de laatste element hebt gehad. Dit heet een conditiegestuurde herhaling (of een sentinel-controlled loop). De situatie en de voorwaarde bepalen dan hoe vaak de code uitgevoerd moet worden. In het voorbeeld van een lijst doorlopen stop je met herhalen als je alle elementen gehad hebt. Hiervoor kun je een while-loop gebruiken. Voor het doorlopen van alle elementen uit een lijst is er een standaard aanpak:
  • Bijhouden waar je bent: maak een variabele positie om bij te houden waar je in de lijst bent.
  • Beginpositie: geeft de variabele positie een beginwaarde, in dit geval 0 (het begin van een lijst is altijd 0)
  • Bepaal de voorwaarde voor herhaling: Dus “Zolang voorwaarde herhaal … “. Dat hangt af van hoe lang de lijst is. Als de lijst 5 elementen heeft, dan moet je het 5 keer doen. Dit kun je berekenen door de lengte van de lijst te bepalen, met len(lijst) De voorwaarde is dus zolang positie nog kleiner is dan len(lijst), met positie < len(lijst).
  • Bepaal wat je steeds wilt herhalen: de acties of berekeningen die steeds per element uitgevoerd moeten worden.
  • Verhoog de variabele: als de actie is uitgevoerd moet de volgende element gepakt worden. Dit doe je door positie met 1 op te hogen.

12.2 Lists zijn veranderbaar

Omdat lists veranderbaar (Engels: "mutable") zijn, mag je de inhoud van een list wijzigen. Om een element van de list te overschrijven, kun je er een nieuwe waarde aan toekennen middels een assignment.

Je kunt ook een sub-list overschrijven door een nieuw list toe te kennen aan de sub-list. De sub-list en de nieuwe list hoeven niet dezelfde lengte te hebben.

Je kunt elementen van een list verwijderen door een lege list aan de te verwijderen sub-list toe te kennen. In Python lees je [1:3] als "op elementen op index 1 tot index 3". In dit geval gaat het over elementen op index 1 (dus de tweede element in de lijst) en index 2 (de derde element in de lijst).

Door sub-lists en assignments te gebruiken, kun je lists aanpassen op iedere manier die je wilt. Het is echter eenvoudiger om lists aan te passen via methodes. Er zijn allerlei handige methodes beschikbaar, die ik hieronder bespreek.

12.3 Lists en operatoren

Lists ondersteunen het gebruik van de operatoren + en *. Deze operatoren werken op eenzelfde manier als ze werken bij strings.

Met de + kun je twee lijsten achter elkaar 'plakken'. Het resultaat is een lijst die de elementen bevat van beide lijsten. Je moet het resultaat aan een variabele toekennen om het op te slaan.

Je kunt een list vermenigvuldigen met een integer om een list te creëren die de elementen van de originele list bevat, net zo vaak herhaald als de integer aangeeft. Dit kan een snelle manier zijn om een list te creëren waarvan alle elementen hetzelfde zijn.

Om een element achteraan een lijst toe te voegen gebruik je de + en zet je het element tussen vierkante haken.

Met append() kun je ook een nieuw element aan het einde van een list plakken. Je roept de methode aan met het nieuwe element als argument.

12.4.2 Verwijder uit een lijst met remove()

remove() laat je een element van de list verwijderen. Het element dat je wilt verwijderen geef je mee als argument. Als dit element meerdere keren voorkomt in de list, wordt de eerste instantie (die met de laagste index) verwijderd. Als je een element probeert te verwijderen dat niet voorkomt op de list, geeft dat een runtime error.
