Afsluitende opgave 6.7: Kwadratische vergelijkingen

Je kunt kwadratische vergelijkingen oplossen met de abc-formule. Kwadratische vergelijkingen hebben de vorm: $$ax^2 + bx + c = 0$$. Dit soort vergelijkingen heeft nul, één of twee oplossingen. Het aantal oplossingen is afhankelijk van de discriminant $$D = b^2 - 4ac$$.
  • Er zijn geen oplossingen als D negatief is.
  • Er is één oplossing als D nul is, namelijk: $$x = \frac{-b}{2a}$$
  • Er zijn twee oplossingen als D positief is namelijk: $$x_{1} = \frac{−b + \sqrt{D}}{2a}$$ $$x_{2} = \frac{−b - \sqrt{D}}{2a}$$
