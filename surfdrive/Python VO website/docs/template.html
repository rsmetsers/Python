Meerdere ifs

Je hoeft je niet te beperken tot slechts één if statement. Je kunt er zoveel hebben als je wilt.

x = 5
if x == 5:
    print( "x is 5" )
if x > 4:
    print( "x is groter dan 4" )
if  x >= 5:
    print( "x is groter of gelijk aan 5" )
if x < 6:
    print( "x is kleiner dan 6" )
if x <= 5:
    print( "x is kleiner of gelijk aan 5" )
if x != 6 :
    print( "x is ongelijk aan 6" )
