Loops

De code om een vierkant te schrijven heeft heel erg veel dezelfde regels code.

def tekenVierkant( lengte ):
    turtle.forward(lengte)
    turtle.right(90)
    turtle.forward(lengte)
    turtle.right(90)
    turtle.forward(lengte)
    turtle.right(90)
    turtle.forward(lengte)
    turtle.right(90)

Je herhaalt namelijk 4 keer:

turtle.forward(lengte)
turtle.right(90)

Kun je nagaan hoeveel regels van dezelfde code je krijgt als je een dodecagon (een figuur met 12 zijden) gaat maken?

Programmeurs gebruiken loops als een oplossing voor stukken code die vaak herhaald worden. Loop spreek je uit als loep. In Python ziet een for loop er zo uit:

for i in range(4):
   # Doe iets

Met deze loop wordt iets 4 keer herhaald door het stukje code: range(4). De code dat herhaalt moet worden moet je laten inspringen.

Als je loops wilt gebruiken bij turtle graphics, dan moet je goed nadenken welke stapjes herhaald moeten worden.

Deze stapjes zet je in de loop op de plek waar in het voorbeeld staat # Doe iets. Je moet ervoor zorgen dat deze regels een klein stukje inspringen, zodat je ook kunt zien dat ze binnen in de loop staan.

Om de figuur te tekenen moet je in de loop de code zetten waarmee turtle naar voren gaat (forward), en de regel code waarmee ze de juiste hoek maakt.
