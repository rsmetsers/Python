Python cursus voor VO (Havo en Vwo)

Welkom bij de website voor Python voor Havo en Vwo. Voor de opdrachten wordt Python 3 gebruikt.

  • Gebruik alleen Google Chrome.
  • Om de opgaven te maken moet je een login bij repl.it maken.
  • Na het inloggen moet je de browser verversen (met F5).
  • Druk op SUBMIT om je antwoorden op te slaan.
